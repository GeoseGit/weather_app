# weather_app
Weather_app- приложение для просмотра погоды, с использованием openweather api.

Мобильное приложение разработанное на flutter, с использованием языка dart.

Для теста на мобильном устройстве, установите app-release.apk (https://github.com/GeoseGit/weather_app/blob/80096d4f7cf12c6b0e90728284c109a758d5f7f7/app-release.apk)
